# theia set

Create .env file

## Usage

```bash
theia set IPADDR [OPTIONS]
```

## Arguments

#### *IPADDR*

IP address of target

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

## Options

#### *--name, -n NAME*

Name of target

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *--hostname, -h HOSTNAME*

Hostname of target

| Attributes      | &nbsp;
|-----------------|-------------
| Needs: | *--extension*

#### *--extension, -e EXTENSION*

URL extension of target

| Attributes      | &nbsp;
|-----------------|-------------
| Needs: | *--hostname*
