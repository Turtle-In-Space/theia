NAME
==================================================

**theia set** - Create .env file

SYNOPSIS
==================================================

**theia set** IPADDR [OPTIONS]

DESCRIPTION
==================================================

Create .env file


ARGUMENTS
==================================================

IPADDR
--------------------------------------------------

IP address of target

- *Required*

OPTIONS
==================================================

--name, -n NAME
--------------------------------------------------

Name of target

- *Required*

--hostname, -h HOSTNAME
--------------------------------------------------

Hostname of target

- Needs: **--extension**

--extension, -e EXTENSION
--------------------------------------------------

URL extension of target

- Needs: **--hostname**
